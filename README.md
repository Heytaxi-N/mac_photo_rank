# 照片转存

一个本地 macOS 小工具，用来把「照片」或 Finder 里的图片和视频批量导出到微店商品图目录，并按点击顺序重命名。

## 使用

1. 运行打包脚本：

   ```bash
   scripts/build_app.sh
   ```

2. 打开生成的 App：

   ```bash
   open dist/照片转存.app
   ```

3. 在 App 里输入文件夹名，例如 `红标短裤`。
4. 点击「添加文件」选择图片或视频，也可以从 Finder 拖入文件；视频建议用「添加文件」选择。
5. 按目标顺序点击缩略图，编号从 `01` 开始；再次点击可取消编号。
6. 如果最后一张是尺码表，勾选「有尺码表」。
7. 点击「导出」。

文件会导出到：

```text
/Users/nick/Downloads/weidian_products-main/商品图/<文件夹名>/
```

图片统一导出为 JPG，视频统一导出为 MP4。未勾选「有尺码表」时，例如：

```text
红标短裤01.jpg
视频01.mp4
红标短裤02.jpg
视频02.mp4
红标短裤03.jpg
```

从「照片」App 直接拖视频时，如果系统只提供预览图，App 会按图片处理；需要导出视频时请用「添加文件」选择。

勾选「有尺码表」时，最后一张图片会命名为：

```text
红标短裤01.jpg
视频01.mp4
尺码表.jpg
```

## 重名处理

如果目标文件夹已经存在，App 会提示：

- 覆盖并重新导出：先清空旧文件夹，再写入新图片。
- 自动新建后缀文件夹：例如 `红标短裤-2`。
- 取消：不做任何导出。
